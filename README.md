## Unturned Map Merge Tool
Program for merging multiple unturned maps into one.  
Work in progress

### Currently Mergeable Files

**Spawns**
* Fauna.dat
* Items.dat
* Jars.dat
* Players.dat
* Vehicles.dat
* Zombies.dat

**Environment**
* Bounds.dat
* Flags.dat
* Flags_Data.dat
* Nodes.dat
* Paths.dat
* Roads.dat

**Terrain**
* Trees.dat

**Level**
* Buildables.dat
* Objects.dat
